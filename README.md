# Chat Agent

This is a basic Chat Agent which is using Google Gemini Large Language Model with OpenAI compatibility

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development.

### Prerequisites

What things you need to install the software and how to install them

- python-dotenv
- openai

These are the python modules that need to be installed.

## Deployment

- Clone the repository
- Add gemini API key
- Run the code
